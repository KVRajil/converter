instructions to run the code

1. install dependencies
    1. rvm version - ruby-2.3.4
    2. gem install pry
    3. gem install prettyprint 
    4. gem install rspec
    5. gem install benchmark
    
2. run the converter
    1. ruby lib/phone_number_to_words.rb
    2. input phone number through console
    
3. run the specs
    1. rspec specs
